# Task Manager (PHP + Slim + SQLite)

Минимальный таск-менеджер с фичей "ии-комментатор антикоуч". API на Slim и интерфейс на чистом HTML/CSS/JS. Хранение задач в SQLite, комментарии генерируются через Gemini при наличии ключа.

## Стек
- PHP 8+
- Slim Framework
- SQLite
- Vanilla JS, HTML, CSS

## Локальный запуск
```bash
composer install
```

```bash
php -S localhost:8080 index.php
```

Открой `http://localhost:8080/`.
